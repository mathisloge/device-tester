# DuT-App

A application to test devices.

# CURRENTLY NOT RECOMMENDED FOR USE 
unless you want a lot of bugs and cumbersome UX.

## Supported connections

Currently DuT-App supports the following connections

* Serial
* TCP-Client
* TCP-Server
* UDP
* COAP-Client
